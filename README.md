# UTRUX

🌟 وب‌سایت رسمی کانال UTRUX

UTRUX یک وب‌سایت شخصی برای نمایش ویدیوهای آموزشی، مدیریت پروفایل و ارتباط با کاربران است. این پروژه با HTML، CSS و JavaScript خالص (Vanilla JS) توسعه داده شده و بدون نیاز به فریم‌ورک اجرا می‌شود.

## ✨ امکانات

* 🎬 نمایش ویدیوهای کانال
* 🔎 جستجوی ویدیوها
* 👤 ساخت و ویرایش پروفایل شخصی
* 💬 ارسال نظر
* 🌍 تغییر زبان
* 🎵 پخش موسیقی پس‌زمینه
* 🧹 ریست پروفایل
* 🚪 خروج از حساب کاربری
* 🔐 مدیریت ویدیوها با رمز عبور
* 💾 ذخیره اطلاعات در LocalStorage

## 🛠️ تکنولوژی‌های استفاده‌شده

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 🚀 نصب و اجرا

1. فایل‌های پروژه را دانلود یا Clone کنید.
2. فایل `index.html` را در مرورگر باز کنید.
3. از امکانات سایت استفاده کنید.

## 🔐 بخش مدیریت

بخش مدیریت برای افزودن و حذف ویدیوها محافظت شده است و نیاز به رمز عبور دارد.
